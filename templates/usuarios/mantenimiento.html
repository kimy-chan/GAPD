Manual de Usuario del Sistema de Inventario

El Sistema de Inventario es una herramienta desarrollada en Python 3.12 y Django 5.0.3 que permite gestionar productos, categorías y existencias. Este manual proporciona instrucciones sobre la instalación, configuración y uso básico del sistema.

2.1. Hardware
  • Procesador: Intel Core i3 o superior
  • Memoria RAM: 4 GB o superior
  • Espacio en Disco: 500 MB disponible
2.2. Software
  • Python 3.12 instalado
  • Administrador de paquetes pip
  • Sistema operativo: Windows, macOS o Linux

3.1. Clonar el Repositorio
  1. Abra la terminal o línea de comandos.
  2. Ejecute el siguiente comando:
    3. git clone https://github.com/usuario/sistema-inventario.git
    cd sistema-inventario
3.2. Crear un Entorno Virtual
python -m venv env

Active el entorno virtual:

  • Windows: .\env\Scripts\activate
  • Linux/Mac: source env/bin/activate
3.3. Instalar Dependencias

Ejecute el comando:

pip install -r requirements.txt
3.4. Configurar la Base de Datos
  1. Realice las migraciones:
    2. python manage.py makemigrations
    python manage.py migrate
3.5. Iniciar el Servidor

Ejecute el comando:

python manage.py runserver

Acceda al sistema en http://127.0.0.1:8000.

Actualización de Dependencias
pip install --upgrade -r requirements.txt
Verificar Errores en Modelos
python manage.py check
Generar Migraciones por Aplicación
python manage.py makemigrations <nombre_app>
Aplicar Migraciones
python manage.py migrate
